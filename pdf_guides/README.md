## Comment générer les PDFs

1. Ouvrir le fichier .html dans Chrome ou Edge
2. Ctrl+P → "Enregistrer en PDF"
3. Format: A4, marges: Aucune, Graphiques d'arrière-plan: activés
4. Cliquer "Enregistrer"

Le bouton "Save as PDF" / "Enregistrer en PDF" en haut à droite fait exactement ça.
